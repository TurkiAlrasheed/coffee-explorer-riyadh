Riyadh Cafes

Each cafe on the map is assigned a score (out of 5) that reflects both its average user rating and its popularity. Here's how the score is calculated:

  1. Normalize both rating and rating_count to a 0-1 range:
    2. rating_norm = (rating - min) / (max - min)
popularity_norm = (rating_count - min) / (max - min)
  2. Combine the normalized values using a weighted average:
    3. score = 5 x (0.9 x rating_norm + 0.1 x popularity_norm)
  3. This gives more importance to the average rating (90%) while still accounting for how many users rated the cafe (10%).

The final result is a score between 0 and 5. Hover over each point on the map to see its name, original rating, number of ratings, and computed score.
